## 游 How to Use Templates

### Quick Start

1. **Navigate to Architecture V3**: `/architecture/v3`
2. **Select Business Context Tab**
3. **Load Templates**:
   - Click "Load TOGAF Entity Templates" for organizations
   - Click "Load TOGAF Stakeholder Templates" for actors
   - Click "Load TOGAF Templates" for capabilities
4. **Select Template Category**
5. **Customize** loaded items to your specific needs

### Best Practices

#### 游꿢 Entity Templates
1. **Start with closest match** - Choose organization type that best fits
2. **Customize locations** - Update with actual geographic locations
3. **Add specifics** - Enhance descriptions with business context
4. **Remove non-applicable** - Delete entities that don't apply
5. **Add custom** - Add organization-specific entities

#### 游논 Actor Templates
1. **Load relevant groups** - Select all applicable stakeholder groups
2. **Map to real people** - Update with actual names when possible
3. **Update activities** - Customize activities to actual roles
4. **Document RACI** - Use for RACI matrix development
5. **Track engagement** - Use for stakeholder engagement planning

#### 游꿢 Capability Templates
1. **Select domains first** - Choose relevant capability domains
2. **Review completeness** - Ensure all critical capabilities covered
3. **Avoid duplicates** - Remove redundant capabilities
4. **Assess maturity** - Document current vs. target maturity
5. **Link to systems** - Map capabilities to supporting applications

### TOGAF ADM Alignment

| TOGAF Phase | Templates to Use | Purpose |
|-------------|-----------------|---------|
| **Preliminary** | All three | Establish architecture framework scope |
| **Phase A: Vision** | Entities, Actors | Define stakeholders and scope |
| **Phase B: Business** | Capabilities, Entities | Business capability assessment |
| **Phase C: Information** | Capabilities (Data-related) | Information architecture scope |
| **Phase D: Technology** | Capabilities (IT-related) | Technology capability mapping |
| **Phase E: Solutions** | All three | Gap analysis and solution options |
| **Phase F: Planning** | Actors | Implementation stakeholder map |
| **Phase G: Governance** | Actors | Governance structure |
| **Phase H: Change** | Capabilities | Benefits realization tracking |
