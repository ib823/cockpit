CRITICAL ISSUES (Blocks Mobile)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. PlanMode Fixed Panel - 480px (PlanMode.tsx:311)
   └─ Problem: Wider than most phones (375px), impossible to close on mobile
   └─ Impact: CRITICAL - Core workflow broken on mobile
   └─ Fix: w-full sm:max-w-sm md:max-w-md lg:w-[480px]

2. PresentMode Fixed Controls (PresentMode.tsx:307-367)
   └─ Problem: Multiple overlapping fixed elements, no safe area awareness
   └─ Impact: CRITICAL - Unusable on phones, controls overlap
   └─ Fix: Redesign layout for mobile, use responsive positioning

3. Gantt Charts - 800-1200px Minimum (GanttChart.tsx, GanttCanvas.tsx)
   └─ Problem: min-w-[1000px] forces excessive horizontal scrolling
   └─ Impact: CRITICAL - Cannot view timelines on mobile/tablets
   └─ Fix: Add vertical task list for mobile, horizontal scroll for desktop

4. AppLayout Horizontal Menu (layout/AppLayout.tsx:42)
   └─ Problem: width: '100vw' causes horizontal scroll, menu doesn't collapse
   └─ Impact: HIGH - Navigation broken on mobile
   └─ Fix: Add hamburger menu, collapse items vertically

✓ Design Tokens System

  └─ Well-organized variables

BAD PATTERNS (What Breaks Mobile)

✗ Hardcoded Pixel Widths (No Responsive Fallbacks)
  └─ w-[480px] (PlanMode)
  └─ w-[240px], w-[280px] (OrgChart)
  └─ min-w-[800px], min-w-[1000px] (Gantt)

✗ Fixed Positioning (No Safe Area Awareness)
  └─ left-12, right-12, top-8, bottom-12 hardcoded
  └─ No env(safe-area-inset-*) for notched devices
  └─ Elements overlap on small screens

✗ No Touch-Specific UX
  └─ No swipe gestures
  └─ No long-press menus
  └─ No haptic feedback

✗ Text Overflow Not Handled
  └─ Missing truncate, line-clamping
  └─ Tab text might break layout ("Implementation Roadmap")

✗ Inconsistent Breakpoint Usage
  └─ xs: defined but never used (320-475px not optimized)
  └─ sm: rarely used
  └─ md: heavily used (768px is too large for some decisions)

✗ Modal Issues
  └─ No max-width on some modals
  └─ max-h-[90vh] problematic with keyboards
  └─ No padding safeguard
